# react webpack boilerplate

A straightforward react 17 with webpack 5 boilerplate

## Features

### Webpack

- webpack
- webpack-cli
- webpack-dev-server

### Loaders

- babel-loader
- eslint-loader

- CSS
  - style-loader
  - css-loader

- SASS (css module enabled)
  - style-loader
  - sass-loader
  - postcss-loader
  - css-loader

### Plugins

- html-webpack-plugin
- mini-css-extract-plugin

## Command

Development

```bash
npm run dev
```

Production

```bash
npm run build
```

Let's it.
